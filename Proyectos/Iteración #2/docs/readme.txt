- Cargar Datos:
    1. Abrir carpeta del proyecto
    2. Dirigirse a la carpeta ".\docs\Scripts"
    3. Abrir archivo "EsquemaAlohandes.sql"
    4. Copiar las queries contenidas dentro del archivo
    5. Pegar en SQLDeveloper
    6. Correr cada una de las queries

- Correr programa:
    1. Abrir carpeta del proyecto
    2. Dirigirse a la carpeta ".\src\main\java\uniandes\isis2304\parranderos\interfazApp"
    3. Abrir archivo "InterfazAlohandesApp.java"
    4. Correr el programa

- Abrir Panel Administrador:
    1. Seleccionar opcion "Operador_Alhandes"
    2. Seleccionar opcion "Entrar como Administrador"
    3. Colocar como usuario "Admin" y como contrase√±a "Admin"

- Crear Operador:
    1. Entrar a panel Administrador
    2. Seleccionar opcion "Registrar operador"
    3. Seleccionar opcion a eleccion de que tipo de operador desea crear
    4. Dar datos y seguir el proceso que solicita la aplicacion

- Crear Cliente:
    1. Seleccionar opcion "Persona_Natural"
    2. Seleccionar opcion "Crear cuenta Cliente"
    3. Ingresar los datos solicitados

- Ingresar como Cliente:
    1. Seleccionar opcion "Persona_Natural"
    2. Seleccionar opcion "Ingresar como Cliente"
    3. Ingresar identificacion de su cuenta

- Cerrar Sesion:
    1. Estar dentro de cualquier panel de Control
    2. Seleccionar opcion "Operacion Cuenta"
    3. Seleccionar opcion "Cerrar Sesion"
